# Paradise Nursery

## Project Overview
Paradise Nursery is an online plant shop application built with React and Redux. Users can:

- Browse a variety of houseplants grouped by categories
- View details such as plant image, name, and price
- Add plants to the shopping cart
- Update cart item quantities
- See the total cost dynamically
- Navigate between Home, Plants, About, and Cart pages

## Technologies Used
- React
- Redux Toolkit
- React Router DOM
- GitHub Pages for deployment
- CSS for styling
